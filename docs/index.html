â“ FAQ

Is my data safe?

Yes! Everything runs locally on your machine. Your league data only comes from the official Sleeper API and never leaves your computer.

Can I customize the stats?

Absolutely! The notebook uses standard SQL (DuckDB dialect), so you can write your own queries against the matchups table.

How often should I run it?

Run it weekly after games complete to get fresh stats. Or run it anytime to check the current state.

What if my league isn't on Sleeper?

This tool only works with Sleeper leagues. The API connects directly to Sleeper's infrastructure.
